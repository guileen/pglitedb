# PGLiteDB Documentation - Japanese

PGLiteDB公式ドキュメントへようこそ。これはPostgreSQL互換の高性能組み込みデータベースです。

## PGLiteDBについて

PGLiteDBは、完全なPostgreSQLワイヤープロトコル互換性を提供しながら、卓越したパフォーマンスを実現する最先端の組み込みデータベースです。CockroachDBのPebbleストレージエンジン（LSMツリーベースのキー値ストア）上に構築されており、PGLiteDBは開発者が愛する使い familiar なPostgreSQLインターフェースを提供し、現代のアプリケーションに必要なパフォーマンス特性を備えています。

ベンチマークでは、2500以上のTPSと4ms未満のレイテンシを実現しており、従来の組み込みデータベースよりも優れたパフォーマンスを発揮しながら、PostgreSQL互換性を維持しています。
