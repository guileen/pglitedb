# PGLiteDB Documentation - Spanish

Bienvenido a la documentación oficial de PGLiteDB, una base de datos embebida de alto rendimiento compatible con PostgreSQL.

## Acerca de PGLiteDB

PGLiteDB es una base de datos embebida de vanguardia que ofrece compatibilidad completa con el protocolo de red de PostgreSQL mientras brinda un rendimiento excepcional. Construida sobre el motor de almacenamiento Pebble de CockroachDB (un almacén de clave-valor basado en árbol LSM), PGLiteDB proporciona la interfaz familiar de PostgreSQL que aman los desarrolladores con las características de rendimiento necesarias para aplicaciones modernas.

Con más de 2500 TPS y menos de 4 ms de latencia en pruebas comparativas, PGLiteDB supera a las bases de datos embebidas tradicionales manteniendo la compatibilidad con PostgreSQL.
